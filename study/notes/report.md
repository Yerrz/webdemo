Uniapp:
https://ask.dcloud.net.cn/article/36484
优点：
1、跨平台应用
2、可使用多种集成开发环境，在微信小程序API的基础上可使用更多跨平台的API和组件库
3、使用Vue.js框架提高了代码重用性
缺点：
1、虽然 uniapp 可以实现跨端开发，但它不能完全屏蔽不同端框架之间的差异，这一点也是开发过程中需要谨慎考虑和解决的地方
2、受到微信小程序平台的限制：微信小程序存在一些功能限制（如网络请求和存储限制），可能会受到平台更改政策的影响，如果某些插件和 API 在未来不再被支持和更新，这可能会对开发的应用造成困扰，dcloud上回应说明uniapp支持直接编写微信原生代码
3、构建复杂的应用可能会出现性能问题：尤其是在处理大量数据和运行内存占用的情况下，会造成页面首次加载缓慢的问题，而且引入大量框架和组件以及内部大多也做了层层封装也会影响性能，dcloud上回应说明uniapp对很多环节做了自动优化，而且微信小程序频繁使用setData代码来更新数据去实现差量数据更新造成视图层和逻辑层频繁进行数据传输更加增加运行的负载
